1697. 检查边长度限制的路径是否存在

Difficulty: 困难

给你一个 n 个点组成的无向图边集 edgeList ，其中edgeList[i] = [ui, vi, disi] 表示点 ui 和点 vi 之间有一条长度为 disi 的边。请注意，两个点之间可能有超过一条边。

给你一个查询数组queries ，其中 queries[j] = [pj, qj, limitj] ，你的任务是对于每个查询 queries[j] ，判断是否存在从 pj 到 qj的路径，且这条路径上的每一条边都 严格小于 limitj。

请你返回一个 布尔数组answer，其中answer.length == queries.length ，当 queries[j] 的查询结果为 true 时， answer 第j 个值为true，否则为 false 。

  • 2 <= n <= 105
  • 1 <= edgeList.length, queries.length <= 105

  • 0 <= ui, vi, pj, qj <= n - 1
  • ui != vi
  • pj != qj
  • 1 <= disi, limitj <= 109
  • 两个点之间可能有 多条 边。
